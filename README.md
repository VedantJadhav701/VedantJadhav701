# 💫 About Me:
👋 Hi, I’m Vedant Jadhav — an AI/ML researcher and engineering student with a strong interest in applying machine learning to physical and real-world systems.

🔭 I’m currently working on AI/ML projects involving **LLMs, RAG pipelines, computer vision**, and **physics- and materials-aware ML systems** (including renewable energy forecasting and metallurgical process optimization).

🌱 I’m actively learning about **physical intelligence**, **embodied AI systems**, and **stochastic modeling for quantitative finance**, with a focus on building models that respect real-world constraints.
